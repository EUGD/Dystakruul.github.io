how to use this:

  1. put the clock code in the input field (or just leave it empty to solve the original one)
  2. click the 'solve' button
  3. copy everything EXCEPT for the readable text
    (e.g. from the result "THERESTVEELHSM" ONLY copy the "VEELHSM" part)
  4. paste that into the input field and repeat
The previous results will also be listed.
When you're at the end, the final solution will be the combination of the first letters of every partial solution.
